PPT如何快捷转markdown文件发布blog

2023/9/13 blog 技术 python markdown

介绍一个包：pptx2md

组会要讲ppt
##### 网站：https://github.com/ssine/pptx2md
需要py3.6以及以上

pip install pptx2md

使用

pptx2md <FILENAME.PPTX>

他可以把ppt里的标题+内容+图片存下来，然后保存为md文件加图片文件夹

我采用的方法是：

  1. 将ppt转为md文件
  2. 将图片上传到图床（picgo）
  3. 替换掉MD文件中的图片链接
  4. 发布！！！！
